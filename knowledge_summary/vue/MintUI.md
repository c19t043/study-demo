# MintUI

## 使用 饿了么的 MintUI 组件

[Github 仓储地址](https://github.com/ElemeFE/mint-ui)

[Mint-UI官方文档](http://mint-ui.github.io/#!/zh-cn)

1. 导入所有MintUI组件：

```
import MintUI from 'mint-ui'
```

2. 导入样式表：

```
import 'mint-ui/lib/style.css'
```

3. 在 vue 中使用 MintUI：

```
Vue.use(MintUI)
```

4. 使用的例子：

```
<mt-button type="primary" size="large">primary</mt-button>
```
